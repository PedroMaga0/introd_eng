O arquivo graphic.py é para gerar o gráfico de audio.wav em função do tempo.

O arquivo learn_audio.py é pra, por enquanto, mostrar algumas informações do áudio (a gente pode fazer algo parecido com isso como feature). 

Tô trabalhando na parte de juntar os canais em um só transformando de Stereo para Mono, mas pretendo usar a mesma biblioteca (wave) que usei no learn_audio.py.


A branch "opt" tem o começo do sistema de opções, onde o usuário digita um número para exibir as informações do áudio e a gente pode pensar em colocar algo do tipo: 1 para gráfico, 2 para informações, 3 para converter, 4 para reproduzir, etc e tal.
